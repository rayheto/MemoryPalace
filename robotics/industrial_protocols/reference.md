# 参考资源

## EtherCAT

- **SOEM (Simple Open EtherCAT Master):** [https://github.com/OpenEtherCATsociety/SOEM](https://github.com/OpenEtherCATsociety/SOEM)
- **IgH EtherCAT Master:** [https://etherlab.org/en/ethercat/](https://etherlab.org/en/ethercat/)
- **Beckhoff Information System:** [https://infosys.beckhoff.com/](https://infosys.beckhoff.com/)
  - 重点看 EtherCAT System Documentation 和 ESC 寄存器定义

## CiA 402

- **CAN in Automation (CiA):** [https://www.can-cia.org/](https://www.can-cia.org/)
  - CiA 402 系列规范: 驱动器和运动控制设备行规

## 其他工业协议

- **CANopen:** CiA 301 (应用层) + CiA 402 (运动控制)
- **Modbus TCP/RTU:** 简单的工业协议，机器人关节常见的调试接口
