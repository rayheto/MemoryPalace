# 电机控制 (FOC) 与传感器融合

针对机械臂或人形机器人，关节模组的底层控制是核心。

## 面试核心考点

### FOC (磁场定向控制)

- **三环级联结构:**
  - **电流环 (最内层):** 转矩控制，最高频率 (10-50 kHz)。通过 PI 控制器调节 Id/Iq 电流
  - **速度环 (中间层):** 速度控制，中频率 (1-10 kHz)。输出为 Iq 电流指令
  - **位置环 (最外层):** 位置控制，低频率 (100-1000 Hz)。输出为速度指令
- **坐标变换:**
  - Clarke 变换: 三相静止 (Ia, Ib, Ic) → 两相静止 (Iα, Iβ)
  - Park 变换: 两相静止 (Iα, Iβ) → 旋转坐标系 (Id, Iq)
  - 逆 Park: 旋转坐标 (Vd, Vq) → 两相静止 (Vα, Vβ)
- **SVPWM (空间矢量脉宽调制):** 六边形矢量空间、扇区判断、占空比计算
- **电流采样:** 单电阻/双电阻/三电阻采样方案的优缺点
- **弱磁控制 (Field Weakening):** 高速区间负 Id 电流注入

### 传感器融合

- **AHRS (航向姿态参考系统):**
  - Mahony 算法 (计算轻量级，适合 MCU)
  - Madgwick 算法 (梯度下降优化，适合 MEMS IMU)
  - EKF (扩展卡尔曼滤波) —— 精度最高但计算量大
- **IMU 数据处理:**
  - 零偏 (Bias) 估计与温度补偿
  - Allan 方差分析噪声特性 (角度随机游走 / 零偏不稳定性)
  - 低通滤波 / 陷波滤波去除电机振动噪声
- **多传感器融合架构:**
  - 视觉惯性里程计 (VIO): 相机 + IMU 紧耦合
  - 编码器 + IMU 互补滤波: 差分运动学 + 惯性数据

### 如何在 MCU 上部署算法

- 定点化 vs 浮点化 (Cortex-M4/M7 带 FPU 可直用浮点)
- CMSIS-DSP 库加速矩阵运算和 FFT
- 轨迹预测与插补在 MCU 上的实现
- 实时性分拆: 高频控制 (FOC 电流环) 放 MCU，低频规划 (轨迹生成) 放 Linux

- **ODrive:** `github.com/odriverobotics/ODrive` —— 高性能伺服驱动开源项目，研究编码器读取和校准流程
- **Madgwick AHRS:** `github.com/xioTechnologies/Fusion` —— 传感器融合参考实现
