# 硬件调试与信号完整性

这是区分"纯软件工程师"和"系统级软硬件工程师"的分水岭。

## 面试核心考点

### 仪器使用

- **示波器 (Oscilloscope):**
  - I2C/SPI 时序抓取与协议解码
  - 上升沿/下降沿时间测量
  - 触发模式设置（边沿、脉宽、逻辑触发）
  - 探头 ×1 / ×10 衰减与带宽选择
- **逻辑分析仪 (Logic Analyzer):**
  - 多通道并行总线解码
  - 协议触发与状态机分析
- **功率分析仪:** 电机功耗评估、效率曲线绘制

### 信号完整性问题

- **信号过冲 (Overshoot) / 下冲 (Undershoot):** 阻抗不匹配引起，用串联端接电阻或降低驱动强度解决
- **串扰 (Crosstalk):** 相邻走线间电容/电感耦合，增加线间距或插入保护地线
- **地线回路 (Ground Loop):** 多点接地电位差导致噪声，采用星型接地或差分信号
- **去耦电容:** 每个 IC 电源引脚附近的去耦电容选值与布局
- **端接匹配:** 串联端接、并联端接、戴维南端接、AC 端接的适用场景

### EMI/EMC 基础

- 辐射发射 (RE) / 传导发射 (CE) 的排查思路
- 屏蔽、滤波、布局走线三大 EMI 控制手段
- PCB 层叠设计（信号层紧邻完整参考平面）
