# 参考资源

## 标准规范

- **GB/T 12642** — 工业机器人性能规范及其试验方法
- **GB/T 2423** — 电工电子产品环境试验 (高低温、振动、湿热)
- **IEC 61508** — 功能安全 (Functional Safety) 基础标准
- **ISO 13849** — 机械安全 — 控制系统安全相关部件
- **IPC-A-610** — 电子组件的可接受性标准

## FMEA

- **AIAG FMEA Manual** (汽车工业行动组 FMEA 手册) — FMEA 方法论标准参考
- **SAE J1739** — 设计/过程 FMEA 推荐实施

## 热设计

- *Hot Air Rises and Heat Sinks* — Tony Kordyban (热设计入门读物)
- **Flotherm / Icepak** — 电子散热仿真工具

## 连接器与线束

- **Molex / TE Connectivity / Amphenol** — 工业/机器人级连接器选型指南
- 关注 IP 等级 (IP67/IP68) 和插拔寿命 (Mating Cycles)
